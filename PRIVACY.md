# Privacy Policy

Just Copy Subtitles for YouTube(TM) does not collect, store, sell, share, or transmit user data.

When the user clicks the extension icon on a YouTube video page, the extension reads caption metadata and subtitle text locally in the browser, formats it, and writes it to the clipboard. No transcript, video URL, browsing history, or personal information is sent to any external server.

The extension does not use analytics, advertising, remote logging, or remote configuration.
